# Milestone 01 Lock — Release Radar MVP

Milestone 01 is complete in 3 sprints.

Release Radar now provides a mobile-first one-page view of monitored TV shows, recent/upcoming episodes, poster artwork, Sonarr download status, and Plex watchability confirmation.

## Locked Scope

- Sonarr calendar and monitored shows
- Poster cards
- Plex confirmation
- Health endpoint
- README and operational docs

## Explicitly Out of Scope

- qBittorrent
- Prowlarr
- auth
- database
- notifications
- show detail pages
- admin panel

## Final MVP Purpose

Answer one question quickly from a phone:

“Is the latest episode out, downloaded, and ready to watch in Plex?”
